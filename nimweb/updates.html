Version 0.3

The changes for the application in latest release are: Version 0.2

  • This release has three strategies included for the computer player.
    1. Cautious Strategy
      In this strategy, the Computer player makes sure to play cautiously by only taking one stick per turn.
    2. Greedy Strategy
      In this strategy, the Computer player makes sure to take maximum number of sticks according to the rules of the game.
    3. Random Strategy
      In this strategy, the Computer player picks up random number of sticks with maximum of 3 and minimum of 1 according to the rules of the game.
  • In this release, the application starts with Cautious Strategy set to the Computer player.
  • The strategy of the Computer player can be changed in the middle of the game or at the start of the game.
  • In this release, we also bring you the option to Exit the game from the Game menu. You can Exit the game at the start or in the middle or at the end.
Version 0.1
  • The first release of the application only allows the Computer player to take one stick per turn.
